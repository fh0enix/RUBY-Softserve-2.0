Завдання 6:
ToDo List (приклад з web: https://www.w3schools.com/howto/howto_js_todolist.asp)
Умови завдання:
Імплементувати необхідні класи для реалізації проекту ToDo List (список завдань, покупок, .....)
Ппотрібна наявність наступних класів (як мінімум)
ToDoItem - для представлення завдання
власне сам ToDoList, який буде якимось чином зв'язаний з ToDoItem
пропонується, щоб для конкретного завдання можна було виставляти ТРИ Статуси:
Active (як тільки задача створена, вона має мати такий статус)
Done (зроблена, вирішена задача)
Deleted (видалена)
